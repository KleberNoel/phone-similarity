# Charsiu Data Form Snapshot

This directory includes a hardcoded CSV snapshot of the language/data-source form
referenced from the CharsiuG2P repository:

- Source sheet: `https://docs.google.com/spreadsheets/d/1y7kisk-UZT9LxpQB0xMIF4CkxJt0iYJlWAnyj6azSBE`
- Source tab gid: `557940309`
- Snapshot file: `charsiu_data_form.csv`

Notes:

- The CSV is a static snapshot for reproducibility and auditability.
- It is informational metadata (language coverage, source types, and licenses),
  not runtime logic.
- The sheet and this repository's `LANGUAGE_CODES_CHARSIU` are not guaranteed to
  match exactly over time.
